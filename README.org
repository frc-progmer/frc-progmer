#+TITLE: FRC初级程序员课程

* 教学目的
  - 掌握c/c++语言的基本概念
  - 独立编写/调试FRC比赛基本程序
  - 比赛中负责电路/气路的搭建和维护

* 基础要求
  - 无

* 学生需要准备的器材/工具
  - 电脑
  - 螺丝刀
  - 斜口钳
  - 电烙铁
